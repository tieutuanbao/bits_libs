Thư viện lập trình LED Matrix HUB12:

1. DP4536, IP4536
- Độ phân giải: 32x16 pixel
- Scan: 1/4
Phương thức quét:
- OE = LOW
- Ghi dịch bit
- LAT = HIGH
- LAT = LOW
- SCAN A, B
- OE = HIGH
